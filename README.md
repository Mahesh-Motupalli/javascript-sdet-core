
# JavaScript Core for SDET

This repository contains core JavaScript concepts implemented with an SDET mindset.
The focus is on clean, reusable, and test-friendly JavaScript rather than UI code.

## Topics Covered
- JavaScript basics and scope
- Functions and closures
- Arrays and objects
- Error handling
- Async fundamentals

## Why This Repo
As an SDET, understanding JavaScript deeply helps in:
- Writing stable automation utilities
- Debugging async test failures
- Designing reusable test frameworks

Each file contains small, focused examples with clarity over complexity.
